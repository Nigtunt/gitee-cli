========================================
Command: gitee 
Usage: gitee [-hV] [--debug] [--json] [-R=<repo>] [COMMAND]
A machine-first CLI for Gitee, mirroring GitHub CLI (gh) commands.
      --debug         Enable debug mode: print detailed request and response
                        info.
  -h, --help          Show this help message and exit.
      --json          Force all output to be valid JSON.
  -R, --repo=<repo>   Target repository in <OWNER>/<REPO> format.
  -V, --version       Print version information and exit.
Commands:
  auth    Manage authentication.
  config  Manage gitee-cli configuration.
  issue   Manage issues.
  pr      Manage pull requests.
  repo    Manage repositories.

Command: gitee issue create
Usage: gitee issue create [-hV] [--debug] [--json] [--assignee=<assignee>]
                          [-b=<description>] [--branch-name=<branchName>]
                          [--due-date=<dueDate>] [--iid=<iid>]
                          [--labels=<labels>]
                          [--plan-started-at=<planStartedAt>]
                          [--priority=<priority>] [-R=<repo>] -t=<title>
Create a new issue.
      --assignee=<assignee>  Assignee ID.
  -b, --body, --description=<description>
                             Description of the issue.
      --branch-name=<branchName>
                             Related branch.
      --debug                Enable debug mode: print detailed request and
                               response info.
      --due-date=<dueDate>   Plan finish time.
  -h, --help                 Show this help message and exit.
      --iid=<iid>            Issue iid. 0 for auto-increment, >0 to set
                               explicitly.
      --json                 Force all output to be valid JSON.
      --labels=<labels>      Comma-separated list of labels.
      --plan-started-at=<planStartedAt>
                             Plan start time.
      --priority=<priority>  Priority: unassigned, low, medium, high, urgent.
  -R, --repo=<repo>          Target repository in <OWNER>/<REPO> format.
  -t, --title=<title>        Title of the issue.
  -V, --version              Print version information and exit.

                        [--page=<page>] [--per-page=<perPage>] [-R=<repo>]

      --debug                Enable debug mode: print detailed request and
                               response info.
      --direction=<direction>
                             Sort direction: asc, desc. Default: desc.
  -h, --help                 Show this help message and exit.
      --json                 Force all output to be valid JSON.
      --page=<page>          Page number. Default: 1.
      --per-page=<perPage>   Results per page. Default: 30.
  -R, --repo=<repo>          Target repository in <OWNER>/<REPO> format.
      --search=<search>      Fuzzy search issues by keyword.
      --sort=<sort>          Sort field: created, updated. Default: created.
      --state=<state>        Filter by state: created, processing, rejected,
                               finished, all. Default: created.
  -V, --version              Print version information and exit.

  -R, --repo=<repo>     Target repository in <OWNER>/<REPO> format.
